# Fast Pizza Restaurant App

Through this app tou choose and order pizzas you like and estimate its arrival time.

### Technologies

- **Vite:** A fast and lightweight build tool that offers a modern development experience with hot module replacement and optimized production builds.

- **React 18:** The latest major release of the popular JavaScript library for building user interfaces.

- **Typescript:** A superset of JavaScript that adds static typing, allowing for improved code quality.

- **Tailwind:** A utility-first CSS framework that provides a set of predefined classes directly in your HTML, making it easier to build consistent designs across various screen sizes.

- **Zod:** A TypeScript-first schema declaration and validation library that helps define, validate, and infer the types of data structures.

## Installation

Project is using `npm` as a main package manager.

```bash
$ npm install
```

## Running the app

```bash
$ npm run dev
```
